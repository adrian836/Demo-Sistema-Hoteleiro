🏨 Hotel Management System (JavaFX)
Sistema completo de gerenciamento hoteleiro desenvolvido para automatizar processos de reservas, check-in e controle de hóspedes. O projeto combina uma interface moderna em JavaFX com uma arquitetura robusta para garantir agilidade no atendimento.
🚀 Funcionalidades

* Gestão de Reservas: Sistema intuitivo para agendamento de estadias.
* Check-in & Check-out: Fluxo simplificado para entrada e saída de hóspedes.
* Controle de Quartos: Visualização em tempo real do status de ocupação (Disponível, Ocupado, Manutenção).
* Cadastro de Hóspedes: Gerenciamento completo de dados e histórico.
* Relatórios Financeiros: (Opcional - se houver) Visão geral de faturamento e despesas.

🛠️ Tecnologias Utilizadas

* Java: Linguagem principal (JDK 17 ou superior).
* JavaFX: Biblioteca para a interface gráfica (UI/UX).
* Scene Builder: Design das telas via FXML.
* Maven/Gradle: Gerenciador de dependências.
* MySQL/PostgreSQL: Banco de dados para persistência (se aplicável).

🔧 Como Executar

   1. Pré-requisitos: Certifique-se de ter o JDK 17+ instalado e o JavaFX SDK.
   2. Clonar o repositório:
   
   git clone https://github.com
   
   3. Configurar o JavaFX: Se estiver usando o VS Code ou IntelliJ, adicione os módulos do JavaFX nas configurações de execução:
   --module-path /caminho/para/javafx/lib --add-modules javafx.controls,javafx.fxml
   4. Rodar a aplicação: Execute a classe Main.java.
